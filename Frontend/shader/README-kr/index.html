    м Җмһҗ мҶҢк°ң

    Patricio Gonzalez Vivo (1982, л¶Җм—җл…ёмҠӨ м•„мқҙл ҲмҠӨ, м•„лҘҙн—ЁнӢ°лӮҳ) лҠ” лүҙмҡ• кё°л°ҳмқҳ м•„нӢ°мҠӨнҠё кІё к°ңл°ңмһҗмқҙлӢӨ. к·ёлҠ” мң кё°м Ғмҡ”мҶҢмҷҖ мқёкіөм Ғмҡ”мҶҢ, м•„лӮ лЎңк·ёмҷҖ л””м§Җн„ё, лӢЁмқјм Ғмҡ”мҶҢмҷҖ м§‘лӢЁм Ғмҡ”мҶҢ мӮ¬мқҙлҘј нғҗн—ҳн•ңлӢӨ. к·ёлҠ” мӮ¬лһҢл“Өкіј н•Ёк»ҳ к°ңл°ңн•ҳкё°мң„н•ҙ мҪ”л“ңлҘј н‘ңнҳ„мҡ”мҶҢлЎң мӮ¬мҡ©н•ҙ мһ‘м—…мқ„ н•ңлӢӨ.

    PatricioлҠ” мӢ¬лҰ¬м№ҳлЈҢ л°Ҹ н‘ңнҳ„мҳҲмҲ мқ„ кіөл¶Җн–ҲлӢӨ. к·ёлҠ” нҢҢмҠЁмҠӨлҢҖн•ҷм—җм„ң л””мһҗмқё л°Ҹ кё°мҲ  м„қмӮ¬н•ҷмң„лҘј ліҙмң н•ҳкі  мһҲкі  к·ёкіім—җм„ң н•ҷмғқмқ„ к°ҖлҘҙм№ҳкі  мһҲлӢӨ. лҳҗн•ң к·ёлҠ” нҳ„мһ¬ Mapzenм—җм„ң к·ёлһҳн”ҪмҠӨ м—”м§ҖлӢҲм–ҙлЎң л§өмқ„ л§Ңл“ңлҠ” мҳӨн”ҲмҶҢмҠӨнҲҙмқ„ к°ңл°ңмӨ‘м—җ мһҲлӢӨ.
