    ејҖе§Ӣ

    д»Җд№ҲжҳҜ Fragment Shader(зүҮж®өзқҖиүІеҷЁ)пјҹ

    еңЁд№ӢеүҚзҡ„з« иҠӮжҲ‘д»¬жҠҠ shaders е’ҢеҸӨи…ҫе ЎеҚ°еҲ·жңҜзӣёжҸҗе№¶и®әгҖӮдёәд»Җд№Ҳиҝҷж ·зұ»жҜ”е‘ўпјҹжӣҙйҮҚиҰҒзҡ„жҳҜпјҢд»Җд№ҲжҳҜ shaderпјҹ

    еҰӮжһңдҪ жӣҫз»Ҹжңүз”Ёи®Ўз®—жңәз»ҳеӣҫзҡ„з»ҸйӘҢпјҢдҪ е°ұзҹҘйҒ“еңЁиҝҷдёӘиҝҮзЁӢдёӯдҪ йңҖиҰҒз”»дёҖдёӘеңҶпјҢз„¶еҗҺдёҖдёӘй•ҝж–№еҪўпјҢдёҖжқЎзәҝпјҢдёҖдәӣдёүи§’еҪўвҖҰвҖҰзӣҙеҲ°з”»еҮәдҪ жғіиҰҒзҡ„еӣҫеғҸгҖӮиҝҷдёӘиҝҮзЁӢеҫҲеғҸз”ЁжүӢеҶҷдёҖе°ҒдҝЎжҲ–дёҖжң¬д№Ұ вҖ”вҖ” йғҪжҳҜдёҖзі»еҲ—зҡ„жҢҮд»ӨпјҢйңҖиҰҒдҪ дёҖд»¶дёҖд»¶е®ҢжҲҗгҖӮ

    Shaders д№ҹжҳҜдёҖзі»еҲ—зҡ„жҢҮд»ӨпјҢдҪҶжҳҜиҝҷдәӣжҢҮд»ӨдјҡеҜ№еұҸе№•дёҠзҡ„жҜҸдёӘеғҸзҙ еҗҢж—¶дёӢиҫҫгҖӮд№ҹе°ұжҳҜиҜҙпјҢдҪ зҡ„д»Јз Ғеҝ…йЎ»ж №жҚ®еғҸзҙ еңЁеұҸе№•дёҠзҡ„дёҚеҗҢдҪҚзҪ®жү§иЎҢдёҚеҗҢзҡ„ж“ҚдҪңгҖӮе°ұеғҸжҙ»еӯ—еҚ°еҲ·пјҢдҪ зҡ„зЁӢеәҸе°ұеғҸдёҖдёӘ functionпјҲеҮҪж•°пјүпјҢиҫ“е…ҘдҪҚзҪ®дҝЎжҒҜпјҢиҫ“еҮәйўңиүІдҝЎжҒҜпјҢеҪ“е®ғзј–иҜ‘е®Ңд№ӢеҗҺдјҡд»ҘзӣёеҪ“еҝ«зҡ„йҖҹеәҰиҝҗиЎҢгҖӮ

    дёәд»Җд№Ҳ shaders иҝҗиЎҢзү№еҲ«еҝ«пјҹ

    дёәдәҶеӣһзӯ”иҝҷдёӘй—®йўҳпјҢдёҚеҫ—дёҚз»ҷеӨ§е®¶д»Ӣз»Қе№¶иЎҢеӨ„зҗҶпјҲparallel processingпјүзҡ„зҘһеҘҮд№ӢеӨ„гҖӮ

    жғіиұЎдҪ зҡ„ CPU жҳҜдёҖдёӘеӨ§зҡ„е·Ҙдёҡз®ЎйҒ“пјҢз„¶еҗҺжҜҸдёҖдёӘд»»еҠЎйғҪжҳҜйҖҡиҝҮиҝҷдёӘз®ЎйҒ“зҡ„жҹҗдәӣдёңиҘҝ вҖ”вҖ” е°ұеғҸдёҖдёӘз”ҹдә§жөҒж°ҙзәҝйӮЈж ·гҖӮжңүдәӣд»»еҠЎиҰҒжҜ”еҲ«зҡ„еӨ§пјҢд№ҹе°ұжҳҜиҜҙиҰҒиҠұиҙ№жӣҙеӨҡж—¶й—ҙе’ҢзІҫеҠӣеҺ»еӨ„зҗҶгҖӮжҲ‘д»¬е°ұз§°е®ғиҰҒжұӮжӣҙејәзҡ„еӨ„зҗҶиғҪеҠӣгҖӮз”ұдәҺи®Ўз®—жңәиҮӘиә«зҡ„жһ¶жһ„пјҢиҝҷдәӣд»»еҠЎйңҖиҰҒдёІиЎҢпјӣеҚідёҖж¬ЎдёҖдёӘең°дҫқеәҸе®ҢжҲҗгҖӮзҺ°д»Ји®Ўз®—жңәйҖҡеёёжңүдёҖз»„еӣӣдёӘеӨ„зҗҶеҷЁпјҢе°ұеғҸиҝҷдёӘз®ЎйҒ“дёҖж ·иҝҗиЎҢпјҢдёҖдёӘжҺҘдёҖдёӘең°еӨ„зҗҶиҝҷдәӣд»»еҠЎпјҢд»ҺиҖҢдҪҝи®Ўз®—жңәжөҒз•…иҝҗиЎҢгҖӮжҜҸдёӘз®ЎйҒ“йҖҡеёёиў«з§°дёәзәҝзЁӢгҖӮ

    и§Ҷйў‘жёёжҲҸе’Ңе…¶д»–еӣҫеҪўеә”з”ЁжҜ”иө·еҲ«зҡ„зЁӢеәҸжқҘиҜҙпјҢйңҖиҰҒй«ҳеҫ—еӨҡзҡ„еӨ„зҗҶиғҪеҠӣгҖӮеӣ дёәе®ғд»¬зҡ„еӣҫеҪўеҶ…е®№йңҖиҰҒж“ҚдҪңж— ж•°еғҸзҙ гҖӮжғіжғізңӢпјҢеұҸе№•дёҠзҡ„жҜҸдёҖдёӘеғҸзҙ йғҪйңҖиҰҒи®Ўз®—пјҢиҖҢеңЁ 3D жёёжҲҸдёӯеҮ дҪ•е’ҢйҖҸи§Ҷд№ҹйғҪйңҖиҰҒи®Ўз®—гҖӮ

    и®©жҲ‘д»¬еӣһеҲ°ејҖе§ӢйӮЈдёӘе…ідәҺз®ЎйҒ“е’Ңд»»еҠЎзҡ„жҜ”е–»гҖӮеұҸе№•дёҠзҡ„жҜҸдёӘеғҸзҙ йғҪд»ЈиЎЁдёҖдёӘжңҖз®ҖеҚ•зҡ„д»»еҠЎгҖӮеҚ•зӢ¬жқҘзңӢе®ҢжҲҗд»»дҪ•дёҖдёӘеғҸзҙ зҡ„д»»еҠЎеҜ№ CPU жқҘиҜҙйғҪеҫҲе®№жҳ“пјҢйӮЈд№Ҳй—®йўҳжқҘдәҶпјҢеұҸе№•дёҠзҡ„жҜҸдёҖдёӘеғҸзҙ йғҪйңҖиҰҒи§ЈеҶіиҝҷж ·зҡ„е°Ҹд»»еҠЎпјҒд№ҹе°ұжҳҜиҜҙпјҢе“ӘжҖ•жҳҜеҜ№дәҺдёҖдёӘиҖҒејҸзҡ„еұҸе№•пјҲеҲҶиҫЁзҺҮ 800x600пјүжқҘиҜҙпјҢйғҪйңҖиҰҒжҜҸеё§еӨ„зҗҶ480000дёӘеғҸзҙ пјҢеҚіжҜҸз§’иҝӣиЎҢ14400000ж¬Ўи®Ўз®—пјҒжҳҜзҡ„пјҢиҝҷеҜ№дәҺеҫ®еӨ„зҗҶеҷЁе°ұжҳҜеӨ§й—®йўҳдәҶпјҒиҖҢеҜ№дәҺдёҖдёӘзҺ°д»Јзҡ„ 2800x1800 и§ҶзҪ‘иҶңеұҸпјҢжҜҸз§’иҝҗиЎҢ60её§пјҢе°ұйңҖиҰҒжҜҸз§’иҝӣиЎҢ311040000ж¬Ўи®Ўз®—гҖӮеӣҫеҪўе·ҘзЁӢеёҲжҳҜеҰӮдҪ•и§ЈеҶіиҝҷдёӘй—®йўҳзҡ„пјҹ

    иҝҷдёӘж—¶еҖҷпјҢе№¶иЎҢеӨ„зҗҶе°ұжҳҜжңҖеҘҪзҡ„и§ЈеҶіж–№жЎҲгҖӮжҜ”иө·з”Ёдёүдә”дёӘејәеӨ§зҡ„еҫ®еӨ„зҗҶеҷЁпјҲжҲ–иҖ…иҜҙвҖңз®ЎйҒ“вҖқпјүжқҘеӨ„зҗҶиҝҷдәӣдҝЎжҒҜпјҢз”ЁдёҖеӨ§е Ҷе°Ҹзҡ„еҫ®еӨ„зҗҶеҷЁжқҘе№¶иЎҢи®Ўз®—пјҢе°ұиҰҒеҘҪеҫ—еӨҡгҖӮиҝҷе°ұжҳҜеӣҫеҪўеӨ„зҗҶеҷЁпјҲGPU : Graphic Processor Unit)зҡ„жқҘз”ұгҖӮ

    и®ҫжғідёҖе Ҷе°ҸеһӢеҫ®еӨ„зҗҶеҷЁжҺ’жҲҗдёҖдёӘе№ійқўзҡ„з”»йқўпјҢеҒҮи®ҫжҜҸдёӘеғҸзҙ зҡ„ж•°жҚ®жҳҜд№’д№“зҗғгҖӮ14400000дёӘд№’д№“зҗғеҸҜд»ҘеңЁдёҖз§’еҶ…йҳ»еЎһеҮ д№Һд»»дҪ•з®ЎйҒ“гҖӮдҪҶжҳҜдёҖйқў800x600зҡ„з®ЎйҒ“еўҷпјҢжҜҸз§’жҺҘж”¶30жіў480000дёӘеғҸзҙ зҡ„дҝЎжҒҜе°ұеҸҜд»ҘжөҒз•…е®ҢжҲҗгҖӮиҝҷеңЁжӣҙй«ҳзҡ„еҲҶиҫЁзҺҮдёӢд№ҹжҳҜжҲҗз«Ӣзҡ„ вҖ”вҖ” е№¶иЎҢзҡ„еӨ„зҗҶеҷЁи¶ҠеӨҡпјҢеҸҜд»ҘеӨ„зҗҶзҡ„ж•°жҚ®жөҒе°ұи¶ҠеӨ§гҖӮ

    еҸҰдёҖдёӘ GPU зҡ„йӯ”жі•жҳҜзү№ж®Ҡж•°еӯҰеҮҪж•°еҸҜйҖҡиҝҮзЎ¬д»¶еҠ йҖҹгҖӮйқһеёёеӨҚжқӮзҡ„ж•°еӯҰж“ҚдҪңеҸҜд»ҘзӣҙжҺҘиў«еҫ®иҠҜзүҮи§ЈеҶіпјҢиҖҢж— йЎ»йҖҡиҝҮиҪҜд»¶гҖӮиҝҷе°ұиЎЁзӨәеҸҜд»Ҙжңүжӣҙеҝ«зҡ„дёүи§’е’Ңзҹ©йҳөиҝҗз®— вҖ”вҖ” е’Ңз”өжөҒдёҖж ·еҝ«гҖӮ
