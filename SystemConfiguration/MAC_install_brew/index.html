    Macе®үиЈ…brew

    1пјҢbrewе®үиЈ…йңҖиҰҒе®үиЈ…xcode-selectпјҢеҰӮжһңжІЎжңүе®үиЈ…xcodeпјҢеҲҷеҸӘйңҖе®үиЈ…xcode-selectпјҡ

    xcode-select --install

    2пјҢе®ҳж–№е®үиЈ…Homebrewж–№ејҸпјҡ

    ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
## йңҖиҰҒзҝ»еўҷпјҢе№¶дё”жҳҜе…ЁеұҖд»ЈзҗҶпјҢеҗҰеҲҷеҸҜиғҪж— жі•иҝһжҺҘеҲ°github

    3пјҢеҸҜиғҪдјҡеҮәзҺ°д»ҘдёӢжҠҘй”ҷдҝЎжҒҜпјҡ

    curl: (7) Failed to connect to raw.githubusercontent.com port 443: Connection refused

    4пјҢжү“ејҖд»ЈзҗҶиҪҜд»¶и®ҫзҪ®пјҢжүҫеҲ°httpд»ЈзҗҶдҝЎжҒҜпјҡ

    5пјҢеңЁcurlдёӯеҠ е…Ҙsocksд»ЈзҗҶең°еқҖпјҡ

    ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install -x 127.0.0.1:1087)"
